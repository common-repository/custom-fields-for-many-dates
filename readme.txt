== Description ==

This plugin will allow a user to add/edit/delete up to 10 custom dates using an easy to use datepicker.

The user will also be able to select a title for each of these dates.  Outputting this data to a wordpress site in a customized 
fashion will take some knowledge of the Wordpress database.

== Installation ==

1. Upload `custom-fields-for-many-dates.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
